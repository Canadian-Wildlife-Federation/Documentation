Interface Components#

The user interface consists of 3 main elements:

  1. the map view window

  2. the legend

  3. the left-hand panel

The Map View Window#

The map space is where data is visualized.

You can zoom in or out of the map space in three different ways: using the scroll button on your mouse, dragging two fingers if using a track pad, or by clicking the â€˜+â€™ or â€˜-â€™ buttons located in the bottom right-hand corner of the map space.

Below the zoom buttons are two other buttons that allow you to toggle between the basemap (left) and satellite imagery (right).

The Legend#

A legend defines the symbology of the data.

Here, the feature type (i.e., dam, waterfall, or fishway) that a data point represents is defined using shape while the passability status of the feature is defined using colour. If you have difficulty distinguishing colours from each other, you can visualize data using the â€˜accessibility modeâ€™. See the topic Visualizing Data for more details.

To gain more space in the map view window, the legend window can be collapsed by clicking on the red circle in the top right corner of the legend window. To restore the legend window, simply click on the â€˜Legendâ€™ button.

The Left-hand Panel#

The left-hand panel is where you can access the bulk of the available web tool functionality.

This functionality includes:

  • Searching for a particular place or structure

  • Visualizing the data in the map view

  • Filtering the data based on a set of criteria

  • Downloading data

Other tabs in the panel will give you access to the CABD documentation site so you can explore the data sources used, review technical documentation, and more.
